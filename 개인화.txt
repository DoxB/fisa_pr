# AI를 활용한 개인화 서비스 

기술 세미나 구성안

* AWS Personalize 서비스 소개 (동작 플로우 이해 및 설명)
* 코드 실행 과정 설명 (가능하면)

1. 행렬 분해의 개념 설명
	1) 행렬의 곱 작동 방식
	2) 숫자의 인수분해를 예시로 들어 메트리스의 분해도 비슷한 과정이라는 것을 표현

2. 각 행열과 값의 의미를 영화 데이터 같은 예시를 들어 설명

3. 분해된 두 메트리스 k값에 대해 설명
	* k는 사용자 선호도와 아이템 특성에 영향을 미치는 숨겨진 요소의 수를 나타냅니다.
	* 예를 들어, 영화 추천 시스템에서, k는 장르, 감독 스타일, 배우 선호도 등과 같은 요소를 나타낼 수 있습니다.
	* 균형 잡기:
		- 더 작은 k: 
			k가 너무 작으면 모델이 지나치게 단순화되어 중요한 세부 사항을 포착하지 못하여 추천의 정확도가 떨어질 수 있습니다.
		- 더 큰 k: 
			k가 k가 너무 크면 모델이 과적합되어 의미 있는 패턴이 아닌 노이즈를 포착할 수 있으며 계산 비용이 많이 들 수 있습니다.

	* 예제를 통한 k 설명

		- 간단한 예제:
		영화를 추천한다고 가정하고 
		k=3. 세 가지 요소는 장르, 인기도, 배우 선호도를 나타낼 수 있습니다. 그러면 각 사용자와 각 영화는 세 개의 숫자 벡터(각 요소에 대해 하나씩)로 표현됩니다.
		- 시각화:
		사용자의 선호도 벡터(예: [0.8, 0.4, 0.7])에 영화의 특징 벡터(예: [0.9, 0.3, 0.5])를 곱하여 예상 평점을 산출하는 시각화를 표시합니다.

4. 머신러닝을 통해 실제값과 예측값을 비교하여 최적화함

5. 최적화된 행렬값을 출력함
